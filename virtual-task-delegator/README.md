# Squad Task Delegator Skill

Esta skill é parte do seu novo **Squad Virtual Corporativo**. Ela foi projetada para organizar a delegação de tarefas entre agentes de IA.

## Como usar

1. Chame esta skill quando tiver um objetivo de alto nível para seu squad.
2. O agente irá decompor o objetivo e sugerir a delegação.

## Exemplo de Uso

"Delegue a criação de uma nova API de autenticação para o Squad de Backend."

## Estrutura da Skill

- `SKILL.md`: Definição técnica da habilidade.
- `evals/evals.json`: Testes de validação para garantir que o agente entenda a delegação.
