# Squad Task Delegator

## Purpose

Use esta skill quando precisar delegar uma tarefa complexa para um ou mais agentes do seu squad virtual. O objetivo é garantir que cada tarefa tenha um dono, um prazo e critérios de aceitação claros, evitando falhas de comunicação.

## Required behavior

O agente deve seguir estes passos antes de confirmar a delegação:

1. **Análise do Objetivo**: Entender o resultado final esperado.
2. **Decomposição**: Quebrar o objetivo em subtarefas menores.
3. **Atribuição**: Identificar qual perfil de agente (ou ferramenta) é melhor para cada subtarefa.
4. **Definição de Done**: Estabelecer o que significa a tarefa estar concluída.

## Workflow

### Passo 1 - Definição da Tarefa
Descreva a tarefa usando o formato:
- **O que**: Ação clara.
- **Quem**: Agente responsável.
- **Critério de Aceite**: Como saberemos que está pronto.

### Passo 2 - Validação de Dependências
Verifique se o agente possui todas as informações e acessos necessários para executar a tarefa.

---
## Output
Um resumo da delegação contendo a lista de tarefas e os respectivos responsáveis.
